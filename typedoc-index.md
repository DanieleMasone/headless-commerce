# Headless Commerce API Reference

This generated reference documents the public TypeScript contracts that support the static commerce demo.

Use it for implementation-level details such as:

- cart reducer, cart context, and cart types
- checkout mock utilities
- product filtering helpers and product model
- formatting and base path routing helpers
- public component props where they form a reusable contract

For the shopper-facing behavior, use the public user guide at `/guide/`.
For architecture, CI, coverage, accessibility, and GitHub Pages trade-offs, use `/engineering/`.
